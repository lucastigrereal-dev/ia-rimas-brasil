# IA de Rimas Brasil - Batalhas de Rap

Sistema de analise e geracao de rimas brasileiras otimizado para ROG Strix G16.

## Recursos
- Analise de rimas de batalhas
- Geracao automatica de rimas por tema
- Interface web responsiva
- API REST completa

## Como usar
1. Acesse o painel web
2. Escolha "GERAR RIMA"
3. Digite um tema (ex: batalha, periferia)
4. Selecione o estilo
5. Clique em "GERAR AGORA"

## Comandos
```bash
npm install     # Instalar dependencias
npm run dev     # Rodar em desenvolvimento
npm run build   # Buildar para producao
npm run preview # Preview da build
```

## Tecnologias
- Hono (backend)
- Cloudflare Pages (deploy)
- TailwindCSS (frontend)
